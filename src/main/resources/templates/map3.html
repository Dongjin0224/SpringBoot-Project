<카카오맵에서 장소 검색하고 좌표(위도, 경도) 알아내기> 카카오맵 API

by 표블로그 tableblog 2020. 7. 9.
반응형

 

 

 

여러 가지 이유로

특정 장소의 좌표(위도, 경도)를 알고자 하시는 분들이 있으실텐데요,

구글맵에서 검색하면 좌표를 얻는 게 어렵지 않기 때문에

구글맵을 많이 활용하지만

제가 실제로 해보니

구글맵에서는 장소 검색이 잘 안되는 경우가 무척 많고,

카카오맵 또는 네이버지도와 상당한 차이가 있어서

주소를 알고도 원하는 위치를 찾기가 쉽지가 않더라구요.

카카오맵 또는 네이버지도와 구글 지도를 동시에 띄워놓고

한참을 뚫어져라 바라보며 비교해야만 합니다. 

 

카카오맵에서 좌표 찾는 방법도 많이 검색해봤는데,

방법이 있다고 하는데 이제는 쓰이지 않는 기능인지

지금은 검색한 대로 하면 좌표가 나오지 않더라구요.

 

그래서

카카오맵 API 를 이용해서

장소를 검색하고, 그 주변을 클릭하면 좌표를 알 수 있도록

직접 만들어봤습니다.

당연히 잘 만든 것도 아니고, 불편한 부분도 많지만

정말 필요하신 분들이 쓸 수 있을 정도는 되지 않을까 싶어 공유하니

조금이나마 도움이 될 수 있었으면 좋겠습니다.

 

원하는 장소를 검색하면

해당 위치로 지도가 이동하며 마커가 생기고,

마커가 아닌 주변을 클릭하면

위도, 경도 좌표 값을 확인할 수 있습니다.

 

 

 

** 이용 방법 예시 **

 

1) 원하는 장소 입력 ex. 서울역

2) 검색하기 클릭

3) 키워드로 검색한 장소에 아래와 같이 마크 표시되면,

그 중 원하는 위치를 클릭하기 (마우스 손바닥)

4) 하단 클릭한 위치의 좌표 (경도, 위도) 확인하기

 

 

 
<키워드로 장소 검색하고, 클릭한 위치 좌표 얻기>
 
 
이용 방법 (자세히)
 
: 장소 입력 - 검색하기 - 위치 클릭 - 좌표 확인
